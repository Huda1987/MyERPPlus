Sistem akan dipasang di server pelanggan atau disetting di server MyERPPlus. User login sistem trial dapat digunakan untuk percobaan input transaksi selama 1 bulan. Jika user cocok atau sudah sesuai dengan kebutuhan perusahaan tersebut, maka user dapat melakukan pembelian/sewa sistem MyERP Plus

TRIAL KEUANGAN

  • 3 user paket keuangan (Finance & Accounting, Fixed Asset) selama 45 hari.
  • Import master data dan saldo awal.
  • Instalasi online di server MyERP Plus.
  • Training dan pendampingan 2 hari secara remote atau kunjungan ke pelanggan.
  • Support online via tiket dan remote Teamviewer selama 45 hari.

Setelah habis masa trial :

  • Lisensi software bisa diperpanjang dengan sistem sewa maupun beli putus.
  • Layanan jasa bisa dipesan sesuai kebutuhan.

Harga belum termasuk :

  • Customize/modifikasi/penambahan software.
  • Biaya transport dan akomodasi untuk kunjungan ke pelanggan.
